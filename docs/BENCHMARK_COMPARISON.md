- 41.3s 可接受（M系列芯片）
- **适用**：无网络 / 无 API key 的本地批量处理

### 🥉 本地快速：qwen2.5vl:3b

- 最快（12.9s），且完全本地
- 但有羊/鹿幻觉风险，qwen3b 偶尔prompt复读（较罕见）
- **适用**：快速预览、不要求高精度的场景

### 🔬 特殊场景：gemma4:e4b

- 叙事风格最具文学感（"时间停滞的氛围"）
